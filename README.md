# tick.fast

Minimal BigTimer-style countdown timer built with Next.js.

## Controls

- **Set**: applies the configured hours/minutes/seconds value.
- **Start**: starts or resumes countdown.
- **Pause**: pauses at current remaining time.
- **Reset**: stops timer and returns display to `00:00:00`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Deploy to Vercel

1. Push this project to GitHub, GitLab, or Bitbucket.
2. In Vercel, click **Add New Project** and import the repository.
3. Keep defaults (Framework Preset: Next.js, Build Command: `next build`).
4. Click **Deploy**.
5. Optionally add your custom domain in project settings.
